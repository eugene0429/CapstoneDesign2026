# YOLO 커스텀 데이터셋 라벨링 가이드

팀원들과 함께 빠르고 간편하게 YOLO 형식의 Bounding Box(BBox)를 칠 수 있는 모듈입니다. 추가적인 복잡한 프로그램 설치 없이, 파이썬 스크립트 하나로 실행 가능합니다.

## 🛠️ 실행 방법

터미널을 열고 `labeler.py` 스크립트를 실행해 주세요.

```bash
# 기본 사용법 (클래스 2개 'bell1', 'bell2' 자동 지정)
python data_collection/labeler.py -i <이미지 폴더 경로>

# 만약 훗날 다른 클래스를 추가하고 싶을 경우 (예: bell1, bell2, person)
python data_collection/labeler.py -i <이미지 폴더 경로> -c "bell1" "bell2" "person"

# 라벨 텍스트 파일을 다른 폴더에 별도로 저장하고 싶은 경우
python data_collection/labeler.py -i <이미지 폴더 경로> -o <저장할 폴더 경로>
```

예시) 바탕화면의 `my_images` 폴더 안에 있는 이미지를 라벨링 하려면 다음과 같이 아주 간단하게 실행할 수 있습니다:
```bash
python data_collection/labeler.py -i ~/Desktop/my_images
```

## 🎮 단축키 및 사용법 (라벨링 창 내부)

스크립트를 실행하면 이미지 창이 뜹니다. 창 안에 단축키 설명이 함께 표시됩니다.

- **`마우스 좌클릭 + 드래그`**: Bounding Box(바운딩 박스) 생성
- **`A` / `D`**: 이전 이미지 / 다음 이미지로 이동. 넘어갈 때 **자동으로 저장**됩니다.
- **`Z`**: 방금 그린 BBox 되돌리기 (실수했을 때 유용)
- **`C`**: 현재 이미지의 모든 BBox 삭제하기
- **`숫자키 0~9`**: 클래스 변경. (예: 0을 누르면 첫 번째 클래스, 1을 누르면 두 번재 클래스 지정)
- **`Q` 또는 `ESC`**: 라벨링 종료 및 저장

## 📝 결과물 형식

라벨링을 완료하면 (혹은 다음 이미지로 넘어가면), 이미지 폴더 내에 이미지와 동일한 이름의 `.txt` 파일이 자동 생성됩니다.

**`.txt` 파일 내부 구조 (YOLO 정규화 포맷):**
`<class_id> <x_center> <y_center> <width> <height>`

*(예시: bell1이 0번, bell2가 1번 일 때, 0번 클래스를 그렸을 경우)*
`0 0.543210 0.345678 0.123456 0.098765` 팀원들이 이 파일과 이미지를 그대로 Ultralytics 등 YOLO 학습에 사용하면 됩니다!

---
**💡 협업 팁:**
- 작업 전에 팀원들과 스크립트 실행 시 전달할 `-c` 파라미터(클래스 목록 및 순서)를 반드시 통일하세요!
- 한 사람이 여러 이미지를 모두 하지 말고, 이미지를 여러 폴더(예: `part1`, `part2`)로 나누어 각자 할당받은 폴더에 대해 진행하면 작업 속도가 배가 됩니다.
